# shootingGAME0

JavaScript で実装したシンプルなシューティングゲーム。ゲーム開発とブラウザ上での描画・入力処理を学ぶための学習用プロジェクトです。

## 概要

ブラウザで動作する2Dシューティングゲームの基礎実装。プレイヤー操作・弾の発射・敵の出現や当たり判定といった、シューティングの基本要素を実装しています。

## 工夫した点

- ゲームループ（更新 → 描画）の基本構造を自前で実装し、フレームごとの状態管理を学習
- 当たり判定やオブジェクト管理など、ゲーム開発の基礎パターンを実践

## 使用技術

- **言語**: JavaScript
- **描画**: HTML5 Canvas（ブラウザ上で動作）

## 遊び方

リポジトリをクローンし、HTMLファイルをブラウザで開くとプレイできます。

> ゲーム開発の基礎を学ぶことを目的としたリポジトリです。
